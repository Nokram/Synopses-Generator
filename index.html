Version 1.1 : Cet outil permet de générer des synopsis d'événements facilement.
Attention : Le synopsis généré en français est différent de celui en anglais. Il ne s'agit pas de sa traduction !

Pour enrichir le générateur :

  1. Rendez-vous sur Gmail et connectez-vous avec votre e-mail Ryzom. (exemple : (votrepseudo)@ryzom.com → inaros@ryzom.com)
  2. Une fois connecté aux services Google, rendez-vous sur ce document.
  3. Entrez vos idées d'action ou de problème dans les colonnes prévues. Si vous êtes anglophone, traduisez vos idées en français. Si vous êtes francophone, traduisez vos idées en anglais.
  4. Rendez-vous sur DeepL si vous avez besoin d'aide pour traduire vos idées.
  5. Vos idées seront ajoutées progressivement.

Pour utiliser le générateur :

  1. C'est tout simple. Cliquez sur le bouton "Générer un Synopsis | Generate a Synopsis".
