Cet outil permet de générer des synopsis d'événements facilement.
Note : Il n'y a pas de traduction de l'un vers l'autre.

Pour enrichir le générateur :

  1. - Créez-vous un compte pour être ensuite en mesure de sauvegarder vos modifications.
  2. - Ne pas toucher au code HTML ni CSS.
  3. - Cliquez sur Collaborate sur le haut de la page pour commencer un codage coopératif.
  4. - Modifier le code Javascript uniquement dans les zones "Dictionnaire" en respectant la forme du code.
  5. - Cliquez sur Save en haut à gauche de la page.

Pour utiliser le générateur :

  1. - C'est tout simple. Cliquez sur le bouton "Générer un Synopsis | Generate a Synopsis".
